## New to rust?

> Complete resources: https://doc.rust-lang.org/book/ch01-01-installation.html

Install it with:
```bash
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

Then go in a project directory and run:
```bash
cargo run
```
